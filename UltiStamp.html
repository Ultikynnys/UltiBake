What Is This?

A projection-based Decal add-on that includes more than 100 presets for most use cases.

Features

  • Preset packs are included, offering more than 100 decals in total. (More added with each update)
  • Full PBR decals that can be applied to any material.
  • Complete Ultibake support for seamless decal baking into textures (useful for exporting to game engines).
  • Create your own preset packs with the add-on for consistent reuse across various Blender projects.
  • Effortless decal creation from any texture or image.
  • Supports even the most complex decals.
  • Add-on support is available for troubleshooting or new suggestions via Discord.

How to Use

Step 1: Initial interface overview.

Step 2: Begin decal creation process.

Step 3: Duplicate or remove decals using keyboard shortcuts.

Step 4: More detailed usage of the interface.

Step 5: Decal creation is performed by clicking the Create Decal button from the Radial Menu using Ctrl+R (by default). Remove decals with X or Del and duplicate them with Shift+D.

Step 6: Load the included preset packs to use them.

Step 7: Move the decal to the top by clicking the "Move Stamp to Front" button.

Step 8: Move decals across materials using the "Move Stamp to Active Material" button.

Step 9: If you break the material, use the "Fix Stamps" button to fix any stamps on the selected object.

Step 10: Final adjustments and confirmations.
