Let's Add a Baking Set!

The most common use case for baking is to bridge whatever you make in Blender with other software using textures. The most commonly used standard for making materials is called PBR(Physically Based Rendering), which is what we are using as well.

Here is how you can bake a simple color texture with the addon.

After you've installed the addon, you'll find it in the 3D-viewport vertical ribbon. Just click the Add button (shown in the image above) and voila! You've created a Baking set. These Baking Sets are handy containers that hold all the parameters you'll need for baking.

Understanding the Basic Parameters

When you expand the Baking set from the Settings button, you'll see the most important parameters for your baking set. For now, all we need to do is select the Low Object as our target. This means we'll be baking all materials on the selected object to a texture.

Just a heads up! The material must contain a Principled BSDF Node for baking to work. That's how we know what we want to bake.

Ready to Bake!

Once the parameters are set, we're ready to Bake! You can do this from either the button on the baking set itself or by baking all available baking sets.

Finished!

Congratulations! You've baked your first texture using Ultibake.

Note: Starting a bake will also open up the folder where the baked texture is made. You can disable this in the addon parameters (same window as installation). Blender will become unresponsive for the duration of the bake, which can vary depending on what is being baked. A good rule of thumb is that doubling the resolution from, for example, 512px -> 1024px, will make the bake take 4 times as long!

Extra Tip

If you want to see the progress of your bake in real-time, you can do so by opening up the System console. This will give you a visual indicator of the progress.
