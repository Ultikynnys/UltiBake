The Baking Sets

Baking sets are containers for information about the bake. It contains the following important information:

  1. Texture name
  2. Bake Path
  3. File format. It's important to not use jpg or png with bakes that utilize the alpha channel, as this will cause problems
  4. Bake type. This defines what input node to use from the bsdf. Hovering over the selection will tell you exactly what input, unless it's a special one like AO or ColorID
  5. Pixel Margin Mode. Either extends the borders or uses the adjacent faces for padding.
  6. Pixel Margin. Extends the baked texture outwards (This should be the distance between UV islands)
  7. Dimensions of texture. You can toggle the fixed aspect ratio to make it non-square.
  8. Cage extrusion. How much to inflate the Low/Cage mesh (useful when using High to Low baking)
  9. Projected ray from high to low distance. If it exceeds the length it will make that projected ray black resulting in a black pixel in the bake.

Note: if a bake set shares the name, path and file format with another bake set. This tells the baker to combine these into a single texture.

10. Bake Units

These contain information about what to use for the baking. You need at least a low object and its UV Map, the rest are optional.

The Isolation button next to low makes sure that nothing affects this object during the bake. This is useful for baking things like Ambient occlusion, as you don't want other objects to cast shadows on your bake in some situations.

Bake Unit Color Channels
