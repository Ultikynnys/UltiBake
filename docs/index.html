UltiBake

UltiBake is a powerful and efficient texture baking tool designed specifically for Blender. It aims to streamline and enhance the texture baking workflow within Blender, providing users with a robust set of tools to achieve high-quality results with ease.

Features

What you can do with UltiBake

Modular Baking Workflow

All the parameters are stored into Baking Sets, which are re-usable and easily configurable to suit any baking situation


They are stored per Scene, so you can have multiple scenes with different baking sets.

Implicit Channel Packing

Channel packing is automatically detected when Bake sets share names, this effectively combines them into a single texture.

PBR baking

Ultibake utilizes the Principled BSDF node for its baking types, this means you bake exactly what you're plugging into the BSDF node

Custom Bake types

Non-default bakes like Ambient Occlusion,Curvature and ColorID are included with the list of BSDF Input bakes

High to Low baking

Individually define what High object goes with which Low object
